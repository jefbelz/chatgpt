# Nano-Banana Photoshop Script Setup

## Why You Need This
The Photoshop script connects to Replicate AI models (Nano-Banana / Flux Kontext). To run the script, you need an **API key** from Replicate ‚Äî a secret code that allows the script to access AI services.

## Step 0 ‚Äî Create a GitHub Account (If Needed)

Replicate uses GitHub for login. Skip this step if you already have a GitHub account.

**Steps:**
1. Go to [https://github.com/join](https://github.com/join)
2. Enter a username, email, and password
3. Complete verification and confirm your email
4. Remember your credentials ‚Äî you‚Äôll need them to sign in to Replicate

---

## Step 1 ‚Äî Log in to Replicate

1. Open [https://replicate.com](https://replicate.com)
2. Click **Sign Up** or **Log In** (top right)
3. Choose **Continue with GitHub** using your GitHub account
‚úÖ You will be redirected to your dashboard

## Step 2 ‚Äî Add Credit

Replicate requires prepaid credits to run models.

1. Click your profile ‚Üí **Account Settings** ‚Üí **Billing**
2. Add **at least $10** via card or PayPal
3. (Optional) Enable **Auto Reload** to keep balance topped up

---

## Step 3 ‚Äî Create API Token

1. In **Account Settings**, open **API Tokens**
2. Click **Create New Token**
3. Copy the token
‚ö†Ô∏è Keep it secret ‚Äî it works like a password

## Step 4 ‚Äî Install Photoshop Script

1. Download the `.jsx` file
2. Copy it into your Photoshop Scripts folder:
   - Windows: `C:\Program Files\Adobe\Adobe Photoshop <version>\Presets\Scripts`
   - Mac: `/Applications/Adobe Photoshop <version>/Presets/Scripts`
3. Restart Photoshop. The script will appear under **File ‚Üí Scripts**

---

## Step 5 ‚Äî Connect the Script to Replicate

1. Run the script: **File ‚Üí Scripts ‚Üí Nano-Banana**
2. Paste the API token from Step 3
3. Save settings ‚Äî the script will remember your key

---

## Step 6 ‚Äî Choose Your Model

1. Open the **Model Selection** dropdown
2. Choose **Nano-Banana** or **Flux Kontext**
3. You are ready to generate images! üéâ

## Troubleshooting

- Forgot GitHub or Replicate login? Recover via their password reset tools
- No balance? Add credits in Billing
- Script not showing in Photoshop? Make sure `.jsx` is in the correct folder and restart Photoshop
