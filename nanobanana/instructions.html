Nano-Banana Photoshop Script Setup

Step 0 — Create a GitHub Account / Шаг 0 — Создание GitHub аккаунта

If you don’t have a GitHub account, create one:

  1. Go to https://github.com/join
  2. Enter username, email, and password
  3. Complete verification and confirm your email
  4. Remember your credentials — you’ll need them to sign in to Replicate

Если у вас нет аккаунта GitHub, создайте его:

  1. Откройте https://github.com/join
  2. Введите имя пользователя, email и пароль
  3. Пройдите проверку и подтвердите email
  4. Запомните свои данные — они нужны для входа в Replicate

Step 1 — Log in to Replicate / Шаг 1 — Вход в Replicate

  1. Go to https://replicate.com
  2. Click "Sign Up" or "Log In" (top right)
  3. Choose "Continue with GitHub" using your GitHub account
✅ You will be redirected to your Replicate dashboard

  1. Откройте https://replicate.com
  2. Нажмите "Sign Up" или "Log In" в правом верхнем углу
  3. Выберите "Continue with GitHub" и используйте свой аккаунт GitHub
✅ После входа вы попадете на Dashboard Replicate

Step 2 — Add Credit / Шаг 1 — Пополнение баланса

  1. Click your profile → Account Settings → Billing
  2. Add at least $10 via card or PayPal
  3. (Optional) Enable auto reload so your balance never runs out

  1. Кликните на свой аватар → Account Settings → Billing
  2. Добавьте минимум $10 через карту или PayPal
  3. (Опционально) Включите авто-пополнение, чтобы баланс не закончился

Step 3 — Create API Token / Шаг 1 — Создание API токена

  1. Go to Account Settings → API Tokens
  2. Click "Create New Token"
  3. Copy the token (keep it secret!)

  1. В Account Settings откройте API Tokens
  2. Нажмите "Create New Token"
  3. Скопируйте токен (храните в секрете!)

Step 4 — Install Photoshop Script / Шаг 2 — Установка скрипта

  1. Download the .jsx file
  2. Copy it into your Photoshop Scripts folder:
    • Windows: C:\Program Files\Adobe\Adobe Photoshop <version>\Presets\Scripts
    • Mac: /Applications/Adobe Photoshop <version>/Presets/Scripts
  3. Restart Photoshop. Script will appear under File → Scripts

  1. Скачайте файл .jsx
  2. Скопируйте его в папку Scripts Photoshop:
    • Windows: C:\Program Files\Adobe\Adobe Photoshop <version>\Presets\Scripts
    • Mac: /Applications/Adobe Photoshop <version>/Presets/Scripts
  3. Перезапустите Photoshop, скрипт появится в меню File → Scripts

Step 5 — Connect Script to Replicate / Шаг 3 — Вставка API ключа

  1. Run the script File → Scripts → Nano-Banana
  2. Paste your copied API token
  3. Save settings — the script will remember your key

  1. Запустите скрипт File → Scripts → Nano-Banana
  2. Вставьте скопированный API токен
  3. Сохраните настройки — скрипт запомнит ключ

Step 6 — Choose Model / Шаг 3 — Выбор модели

  1. Open Model Selection dropdown
  2. Choose Nano-Banana or Flux Kontext
  3. Ready to generate images!

  1. Откройте выпадающий список Model Selection
  2. Выберите Nano-Banana или Flux Kontext
  3. Готово к генерации изображений!
